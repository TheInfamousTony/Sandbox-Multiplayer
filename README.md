# Sandbox Multiplayer
a WIP sandbox game. Still in alpha stage. Created to have fun with friends (:  
To play: download the "Build" folder and open the exe file (not the UnityCrashHandler64).  
To open the project: download the whole folder and open it in Unity (version 6000.3.18+).  

Instructions(because there are no instructions in the game yet🥲):  
The first menu: enter your name  
The second menu: enter the server name to create, or join an existing server on the list (can test by opening multiple windows of the game)  
The third menu: choose skin and see other people in the server
W, A, S, D keys for movements  
E to open inventory, press on items to select them (ignore the tooltip)  
There are 3 types of items in the inventory: blocks | entities | tools  
- when selecting a block: right-click to place, left-click to delete  
- when selecting an entity (physics objects): right-click to spawn the entity, left-click to delete, hold middle-click to drag entities  
- when selecting a tool: left-click to use  
Esc to pause the game (will not pause the server)  
can change the settings (sensitivity, resolution, and toggle fullscreen) in the pause menu  
G to emote (cringe alert)  
How to use paper blocks: place paper blocks to create a canvas, and use the brush to draw on it
